# Conversor de Binário para Decimal

Projeto simples desenvolvido com **JavaScript**, **HTML** e **Bootstrap** para converter números binários em números decimais.

---

## 🚀 Funcionalidades
- Entrada de números binários
- Conversão automática para decimal
- Interface simples e responsiva utilizando Bootstrap
- Validação básica de entrada

---

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript

---

## 📚 Conhecimentos Aplicados
Este projeto foi desenvolvido com base nos conhecimentos adquiridos nas disciplinas:

- Introdução à Computação
- Desenvolvimento Web Front-End
- Algoritmos e Técnicas de Programação

---

## 📌 Objetivo
Praticar lógica de programação, manipulação do DOM e desenvolvimento de interfaces web simples.
